# ichux-lista-10
-Equipe : Caua Furst, João Magalhães, Luiz Guilherme

-Criamos um apicativo semelhante ao ifood para otimizar o tempo de espera na fila.

Tela 1 (Login): O usuário insere nome e senha para acessar o sistema.
Tela 2 (Busca de estabelecimentos): Permite procurar restaurantes próximos e selecionar uma opção (ex: McDonald’s, Burger King).
Tela 3 (Escolha do pedido): O usuário escolhe os itens do cardápio (como hambúrgueres) e pode confirmar ou cancelar a seleção.
Tela 4 (Confirmação): Mostra o item selecionado e oferece a opção de adicionar mais itens antes de finalizar o pedido.
Tela 5 (Acompanhamento): Exibe o status do pedido, tempo estimado de entrega, posição na fila e opção de avaliar o restaurante.

O fluxo principal é a sequência de ações essenciais que o usuário realiza para completar a função principal do sistema.
